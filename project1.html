Project 1: Rakshik

COVID Protective Gear for Frontline Warriors

Overview

"Rakshik" is a smart wearable band designed specifically for frontline warriors such as doctors, health officials, and cleaning staff who were tirelessly dedicated to saving lives during the COVID-19 pandemic.

The band is equipped with sensors to monitor vital health parameters, including heartbeat and oxygen levels (SpO2), providing real-time data to ensure the well-being of these essential workers.

Features

  • Real-Time Health Monitoring: Continuously tracks heartbeat and oxygen levels, alerting users to any anomalies.
  • User-Friendly Interface: Easy-to-read display showing current health metrics.
  • Durable Design: Built to withstand rigorous use in medical and cleaning environments.
  • Long Battery Life: Ensures the band remains functional throughout long shifts.
  • Data Logging: Stores health data for trend analysis and historical reference.
  • Emergency Alerts: Sends alerts to designated contacts if critical health thresholds are crossed.

Technologies Used

  • Hardware: Custom-designed wearable with integrated sensors for heartbeat and SpO2 monitoring.
  • Microcontroller: Low-power microcontroller to manage sensor data and interface with the user display.
  • Bluetooth: For data synchronization with mobile or cloud applications.
  • Mobile Application: Companion app for Android and iOS for data visualization, alert configuration, and health trend analysis.
  • Cloud Integration: Secure cloud storage for health data with analytics capabilities.

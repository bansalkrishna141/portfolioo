Overview

The "Advanced Security System" is a cutting-edge surveillance project designed to enhance security by detecting unauthorized motion in a room. It sends an SMS alert and a photo of the intruder to the homeowner.

Additionally, the system is trained to recognize specific faces, and if a recognized person enters the room, it sends a personalized message identifying the individual. This system uses motion detection, facial recognition, and real-time communication technologies to provide a robust security solution.

Features

  • Motion Detection: Uses sensors to detect any movement within the monitored area.
  • SMS Alerts: Sends an immediate SMS notification to the homeowner when motion is detected.
  • Intruder Photo Capture: Captures a photo of the person entering the room and sends it to the homeowner.
  • Facial Recognition: Identifies pre-trained, selected faces and sends a personalized message when these individuals enter the room.
  • Real-Time Monitoring: Provides real-time monitoring and alerts to ensure immediate response.

Technologies Used

  • Raspberry Pi: Serves as the central processing unit for running the security software.
  • PIR Motion Sensors: Detects motion within the room.
  • Camera Module: Captures photos and video of the detected motion and intruders.
  • Facial Recognition Software: Machine learning algorithms trained to recognize specific faces.
  • SMS Gateway
  • Cloud Storage: Stores captured photos and videos securely.
